- 1단계: GPIO 기본 (입출력 제어)


- 가장 기본이 되는 디지털 신호 제어입니다.
- 실습 내용: blink_1LED와 toggle_LED 구현하기
- 핵심 레지스터: DDRx, PORTx, PINx
- 학습 목표: * 특정 비트만 조작하는 비트 마스킹(|=, &= ~) 익히기
- 내부 풀업 저항을 레지스터로 활성화하여 스위치 입력 받기
